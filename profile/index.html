スキル

研究

研究テーマ：Neural Architecture Search(NAS)の効率化
現在ニューラルネットワークを活用するために必要な，データ前処理からモデルの構築・評価までの自動化を目指したAutomated Machine Learning(AutoML)と呼ばれる研究が進められています． 私はこの中でもニューラルネットワークの構造を探索する手法であるNASの効率化を目指しています．

学士論文題目「新たなZero-shot CNN評価指標の作成」

関連研究分野

  • NAS：ZiCo，Zen-NAS， NASWOT， DARTS， ENAS等
  • AutoML：データの前処理・選定・加工， モデルの構築，モデルの評価（全て自動で行う）
  • 深層学習：CNN，ResNet等
  • 画像分類問題：MNIST，Cifar-10等
  • モデル圧縮：プルーニング，宝くじ仮説
  • ニューラルネットワークの表現力
