スキル

研究

Network Architecture Search (NAS)に関する研究を行なっている．
NASはニューラルネットワークの構造を自動的に探索する手法である． この手法は膨大な計算量が必要であることが問題として挙げられている． この問題を解決するため，分類性能の評価をZero-Shotで行うことを目指している．

学士論文題目「新たなZero-shot CNN評価指標の作成」

関連研究分野

  • NAS：Zen-NAS， NASWOT， DARTS， ENAS等
  • AutoML：データの前処理・選定・加工， モデルの構築，モデルの評価（全て自動で行う）
  • 深層学習：CNN，ResNet等
  • 画像分類問題：MNIST，Cifar-10等
  • モデル圧縮：プルーニング，宝くじ仮説
  • ニューラルネットワークの表現力
